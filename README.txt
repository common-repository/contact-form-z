== Description ==

Contact Form Z — free contact form plugin for WordPress sites. Powerful & simple. Absolutely free!

The plugin allows you to easily create and place contact forms for text messages from users, pre-orders and bookings, other feedback forms on the site.

== Installation ==

1. Sign in to your WordPress website (your_site.com/wp-admin/).
2. Go to 'Plugins' -> Press 'Add new'.
3. Select 'Keyword' next to the search field in the top right corner and type 'ZAD'. Press Enter.
4. You will see a block with the ‘Contact Form Z’ plugin; click Install.
5. Once the plugin installation is complete, the Activate button will appear. Click this button.
6. After a successful installation and activation, you will see a new ‘Contact Form Z’ menu item in WordPress.
7. The plugin is ready!
